# sutras

**sutras** is a tool for seeing how computer science concepts and algorithms actually work. It turns dry code into a step-by-step visual experience, making it easier to understand what's happening under the hood.

## 🛠 How it Works

Sutras gives you full control over the simulation so you can learn at your own pace:

* **Step-by-Step Moves:** Instead of watching a fast animation, you can click through the algorithm one small change at a time. This lets you see exactly how the data shifts.
* **Speed & Pause:**
  * **Adjustable Speed:** Slow it down to a crawl to study the details, or speed it up to see the big picture.
  * **Play/Pause:** Stop the action at any point to look at the current state of the system.
* **Your Own Data:**
  * **Manual Input:** Type in your own text, numbers, or patterns to see how the algorithm handles them.
  * **Random Mode:** Let the tool create random data for you to test different scenarios instantly.
